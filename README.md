# مشہور - لینڈنگ پیج

پاکستان کا مشہور ای کامرس پلیٹ فارم - لینڈنگ پیج

## شروع کرنے کا طریقہ

```bash
# Dependencies install کریں
npm install

# Development server چلائیں
npm run dev
```

## ٹیکنالوجی

- **Next.js 14** - React Framework
- **TailwindCSS** - Styling
- **Framer Motion** - Animations
- **Lucide React** - Icons
- **TypeScript** - Type Safety

- ✅ Responsive Design (Mobile + Desktop)

- ✅ Modern UI/UX
- ✅ Smooth Animations
- ✅ Glass Morphism Effects
